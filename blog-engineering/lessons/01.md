# 拒绝 git 三板斧

> git add . | git commit -m "commit" | git push

## 初始化及状态查询

```bash
# 初始化一个git项目
git init
# 查看状态
git status
git status -sb
# 查看修改
git diff
```

### 撤销指定 add

```bash
# 撤销指定文件的add
git rm --cached [file-name]
git reset HEAD  [file-name]
# 撤销全部add
git reset HEAD .
```

### 修改 commit 描述

```bash
# 修改commit描述
git commit -m --amend
```

### 版本回退

```bash
# 重置当前分支的HEAD为指定commit，同时重置暂存区和工作区，与指定commit一致 commitid可以通过git log查看
git reset --hard [commitid]
```
