# 基于 nuxt 的个人博客系统 `客户端`

## 技术栈

`vue+nuxt+less+es6+webpack`

### 使用步骤

1. 先要开启[后端服务](https://github.com/dirkhe1051931999/vue-blog-management)
2. 再开启前端服务

## 完成的功能

1. 文章列表，文章详情
2. 分类列表，分类详情
3. 标签列表，标签详情
4. 项目列表，项目详情
5. 搜索列表，搜索详情
6. 留言板
